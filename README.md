LE Mans Ultimate – DLC UNLOCKER
===========================================================

 Lightweight installer for unlocking DLC content for LMU

-----------------------------------------------------------
 FEATURES
-----------------------------------------------------------

  • One-click installation
  • Clean & minimal interface
  • Automatic directory detection

-----------------------------------------------------------
 REQUIREMENTS
-----------------------------------------------------------

  • Installed copy of Le Mans Ultimate (must be steam version)

-----------------------------------------------------------
 INSTALLATION
-----------------------------------------------------------

  1. Download the LATEST RELEASE on github from ACCC.
  2. Extract the archive.
  3. Run the executable file.

  When the program opens:

     → Select your Assetto Corsa Competizione folder
     → Click INSTALL
     → Wait for completion

-----------------------------------------------------------
 DEFAULT GAME LOCATION (STEAM)
-----------------------------------------------------------

  C:\Program Files (x86)\Steam\steamapps\common\Le Mans Ultimate

-----------------------------------------------------------
 USAGE
-----------------------------------------------------------

  • Launch the game normally.
  • The hack will automatically load.

-----------------------------------------------------------
 TROUBLESHOOTING
-----------------------------------------------------------

  Installation fails:
    - Verify the selected folder
    - Run as Administrator
    - Check antivirus interference

  Content not appearing:
    - Restart the game
    - Re-run installer & confirm path
